# Netflix Movie Data Analysis

## 📌 Overview
This project analyzes Netflix movie data to extract meaningful insights and trends using data analysis techniques.

## 🚀 Key Objectives
- Identify most frequent movie genres
- Analyze popularity and voting trends
- Find highest and lowest rated movies
- Explore yearly movie release patterns

## 🛠 Tools Used
- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  

## 📈 Outcome
Provided data-driven insights to support decision-making in content strategy.
